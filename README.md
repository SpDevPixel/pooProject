# 🚽 화장실 급할 땐? (Where is Toilet?)

> **"당황스러운 순간, 당신의 가장 가까운 안식처를 찾아드립니다."**
> 공공 데이터와 사용자 참여형 정보를 결합한 위치 기반 화장실 탐색 웹 서비스.

---

## 📝 프로젝트 소개 (Project Overview)
외출 중 급하게 화장실을 찾아야 하는 상황은 누구나 겪는 당혹스러운 순간입니다. 기존 공공 데이터만으로는 실시간 청결도나 폐쇄 여부를 파악하기 어렵다는 한계를 극복하고자 본 프로젝트를 기획하게 되었습니다.

"화장실 급할 땐?"은 현재 위치를 기반으로 가장 가까운 화장실 정보를 즉각 제공하며, 사용자의 피드백(청결도, 운영 시간, 도어락 유무 등)을 통해 데이터의 신뢰도와 현행성을 실시간으로 유지합니다.

### ✨ 핵심 특징 (Key Features)
* **최소 조작 UX**: 급박한 상황에서 여러 번 클릭하지 않고도 가장 가까운 화장실 경로를 즉시 확인 가능하도록 설계되었습니다.
* **사용자 참여형 데이터**: 사용자가 직접 등록하는 평점, 청결도, 시설 상태를 통해 실시간 데이터를 검증합니다.
* **상황 맞춤형 필터링**: 장애인 화장실, 편의시설 포함 등 개별 사용자의 선호도를 반영한 검색 결과를 제공합니다.
* **기술적 차별성**: 카카오맵 API 연동을 통한 최적 경로 안내 및 DB 연동 마커 표시 기능을 제공합니다.

---

## 📌 주요 기능 (Functional Requirements)

### 🗺️ 지도 및 위치 서비스
* **지도 UI**: 카카오맵 API를 활용한 지도 렌더링 및 실시간 위치 표시.
* **스마트 길 안내**: 사용자 위치 기반 최적 경로 및 평점순 화장실 경로 안내.
* **데이터 마커**: DB에 저장된 화장실 데이터를 지도상에 마커로 시각화.

### ✍️ 화장실 및 리뷰 관리
* **화장실 등록/삭제**: 새로운 화장실 정보를 등록하거나 잘못된 정보 및 폐쇄된 화장실 삭제.
* **리뷰 시스템**: 평점, 청결도, 시설 확인(휴지, 도어락 등) 및 상세 리뷰 작성/수정/삭제.
* **데이터 동기화**: 관리자가 공공 API 데이터를 로컬 DB와 수동 동기화.

### 👤 사용자 서비스
* **회원 관리**: 회원가입, 로그인/로그아웃 및 회원 정보 수정/탈퇴.
* **마이페이지**: 활동 내역(등록한 화장실, 리뷰 목록) 및 회원 정보 확인.
* **즐겨찾기**: 만족스러운 화장실 정보를 저장하여 관리.
